This is a web based android application launcher.
==============
This framework includes 
--------------
>Application management system:
>	1. Add applications
>	2. remove applications
>	3. update applications (releated info of apps, position management of the apps) 
>	4. application's paging system.
>	5. launch apps (under developement)
>Widget management system:
>	1. Add widgets
>	2. remove widgets
>Event system:
>	Normal:
>		1. touchstart
>		2. touchmove
>		3. touchend
>		4. click
>	special:
>		1. longtap
>		2. dbclick
>		3. orientationchange
>		4. resize
>		5. cache releated
>Notifacation system:
>	under developement.
